📊 Task 3: Dashboard Development
📌 Project Overview

This project focuses on creating an interactive dashboard to visualize and analyze a dataset using Power BI. The dashboard provides meaningful insights into Amazon product reviews, helping users understand trends, customer sentiment, and product performance.

🎯 Objective

To design and develop a fully functional interactive dashboard using tools like Tableau, Power BI, or Dash that enables data-driven decision-making through clear and engaging visualizations.

🛠️ Tools & Technologies
Power BI 📊
Data Visualization Techniques
Data Cleaning & Transformation

📊 Dashboard Features
Rating Distribution: Visual representation of product ratings (1–5 stars)
Sentiment Analysis: Breakdown of positive vs negative reviews
Time-Based Trends: Review activity over time
Top Products: Most reviewed products
Filters/Slicers: Interactive filtering based on score, sentiment, and product ID
KPIs:
Average Rating
Most Reviewed Product

📈 Key Insights
Majority of reviews are positive, indicating good customer satisfaction
Higher ratings (4 & 5 stars) dominate the dataset
Certain products receive significantly more reviews, showing popularity
Review activity has increased over time
